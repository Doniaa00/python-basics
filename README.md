# python basics projects

this repository contains beginner python projects created to practice essential programming skills such as variables, loops, conditions, functions, and input/output.

each project is in its own folder with a dedicated readme explaining its purpose and how to run it.

---

## projects

### 1. [calculator](calculator/README.md)
a basic calculator that can add, subtract, multiply, and divide numbers.

### 2. [number guessing game](guessing-game/README.md)
a simple command-line game where you guess a random number between 1 and 100. provides feedback if the guess is too high or too low.

### 3. [password strength checker](password-checker/README.md)
an interactive program that checks the strength of your password based on length, lowercase, uppercase, digits, and special symbols.

---

1. clone the repository:

```bash
git clone https://github.com/Doniaa00/python-basics.git

2.navigate to the project folder you want to run, for example:
