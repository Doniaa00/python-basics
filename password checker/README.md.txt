# Password Strength Checker

Check the strength of your password with this interactive Python program.  
It evaluates passwords based on **length, lowercase and uppercase letters, digits, and special symbols**, and provides clear feedback.

---

## Features

- Interactive command-line program  
- Checks multiple passwords without restarting  
- Tells you if your password is **weak**, **medium**, or **strong**  
- Handles invalid input gracefully  
- Friendly user experience

---

## Skills Demonstrated

- Variables and data types  
- Strings and string methods  
- Loops and conditions  
- Input/Output  
- Basic logic and scoring system

---

1. Make sure Python is installed on your computer.
